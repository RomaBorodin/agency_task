# Agency task

Мой сайт-задание созданный с помощью **flask**

---

## Функционал:

- Список всех агентов (`/`)
- Добавление нового агента (`/add`)
- Все данные о конкретном агенте (`/agent/<id>`)
- Редактирование данных агента (`/edit/<id>`)
- Удаление агента из базы (`/delete/<id>`)

---

## Структура базы данных:

- ID
- Кодовое имя
- Контактный номер
- Email
- Уровень доступа
